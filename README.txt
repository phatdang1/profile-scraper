Profile-Scraper

Description:
This is a Python base project.
It starts by launching Selenium web browser and login to LinkedIn account and navigate to the company below’s profile page？
“, then to the employee section of the company profile page, and scrape the following information for 200 random employees of this company, and then aggregate the info in a tabulated CSV/XLS (for the csv data). 

Company: Hubspot (https://www.linkedin.com/company/hubspot/):

Info to be collected: 
First name 
Last name
Label (the designation right under someone’s name in Linked in)
Location
Profile picture

Run project:
in src folder run the main file scraper.py
